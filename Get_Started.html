Installation

Download ASUN and kick in for your projects.
Download, Extract and link to HTML file  

Install Via Npm
Install ASUN in your Node.js powered apps with the npm package:
1 $ NPM Install ASUN.CSS~GoatMEAT

Install with Ruby
Install ASUN in your Ruby apps using Bundler (recommended) and RubyGems by adding the following line to your Gemfile:
gem 'Asun', '~> 1.13'

Alternatively, if youâ€™re not using Bundler, you can install the gem by running this command:
gem install Asun -v 1.13

You can also Install and Manage ASUN's SASS using Composer
$ composer require twbs/asun:1.13.0

Use Via CDN
Copy and link into head of your html file

NuGet
If you develop in .NET, you can also install and manage ASUN's CSS and SaSS files using NuGet:
PM> Install-Package asun

PM> Install-Package asun.sass
